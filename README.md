# 聲明
1. 因為遊戲裡UI設計太差，許多操作步驟需要非常繁雜操作，而且還需要重複操作幾千次甚至上萬次，動輒你就需要消耗幾小時去手動操作。因此此專案才會設計來解決各種問題。
2. 跟官方反應幾忽是無解，大概我們這種小眾沒有話語權吧。
3. 但是我不製作任何關於練功，自動喝水、自動Buff等破壞遊戲平衡的程式。
